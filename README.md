# Expose files 

> Small project to watch a directory and expose new created files via http(s?).

## How to install
It requires `npm` and `node`.
```
$ git clone https://github.com/a-peyrard/expose-files.git
$ npm install -g ./expose-files
```

## How to run
```
$ expose-files
```
